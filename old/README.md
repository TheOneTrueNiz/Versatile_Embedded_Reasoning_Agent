# old/

Archived material that is not part of the pushable VERA harness surface.

Structure:
- `docs/`: historical or one-off artifacts
- `project_context/`: higher-level product/mission documents not needed for runtime harnessing
- `roadmaps/`: older upgrade plans and execution roadmaps
- `local/`: machine-local leftovers, generated artifacts, and retired harness material

`old/local/` is ignored and should not be pushed.
